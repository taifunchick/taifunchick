## 🚀 Избранные проекты

### 🦅 Cyber Bird – аркадный клон Флаппи-бёрд
*Unity 2022.3, C#, PluginYG, Singleton*  
Мобильная игра на подобии Flappy Bird с бесконечным режимом с красивой графикой. Реализована система скинов птиц, труб и заднего фона, настройки музыки и звука, интеграция с Яндекс.Играми через плагин PluginYG.  
➡️ [Репозиторий](https://github.com/taifunchick/CyberBird)

### 🔫 Ragefront – топ-даун шутер
*Unity 2022.3, C#, ScriptableObject*  
Динамичный топ-даун шутер, где вы оказываетесь в эпицентре бесконечных волн врагов. Раелизована система смены оружия, волн врагов.
➡️ [Репозиторий](https://github.com/taifunchick/Ragefront)

### 🧪 [Название проекта 3] – прототип RPG с открытым миром
*Unity 2021, C#, Terrain система, Behavior Trees*  
Экспериментальный проект для изучения больших ландшафтов, систем квестов и поведения NPC на основе деревьев поведения.  
➡️ [Репозиторий](https://github.com/username/project3)
